#Quandl API wrapper in Clojure 
===================================

> A simple wrapper for interacting with [Quandl API](https://www.quandl.com/docs/api)

## Get it
```clojure
[clj-quandl-api "0.1.0"]

;; In your ns statement:
(ns my.ns
  (:require [clj-quandl-api.core :refer :all]))
```
